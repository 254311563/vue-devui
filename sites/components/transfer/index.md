### API

|    参数     |   类型   |   默认    | 说明                     | 跳转 Demo                         | 全局配置项 |
| :---------: | :------: | :-------: | :----------------------- | --------------------------------- | --------- |
|    id     | `string` |    --     | 可选，文本框 id          | [基本用法](#基本用法)             ||
