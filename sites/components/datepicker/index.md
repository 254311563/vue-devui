# DatePicker 日期选择器

日期、时间的可视化输入。

### 何时使用

- 可视化操作日期选择
- 选择日期区间
- 限制日期输入

### 日期选择

```jsx
<d-datepicker />
```

<d-datepicker />

### 日期区域选择

```jsx
<d-datepicker type="range" />
```

<d-datepicker type="range" />

### 自动关闭

```jsx
<d-datepicker auto-complete />
<d-datepicker auto-complete type="range" />
```

<d-datepicker auto-complete />
<d-datepicker auto-complete type="range" />
